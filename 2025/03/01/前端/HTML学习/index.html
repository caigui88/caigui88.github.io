文档说明

基本概念

  • tag 标签
  • attribute 属性:对标签特征进行设置的一种方式,一般在开始标签中定义
  • text 文本:双标签内的文字
  • element 元素: 开始标签+属性+文本+结束标签→一个元素开始标签+属性+文本+结束标签 \rightarrow 一个元素开始标签+属性+文本+结束标签→一个元素

语法细节

  1. 根标签只有一个
  2. 单/双标签需要正确关闭(为了浏览器可以顺利识别)
  3. 标签可以嵌套但不可以交叉嵌套——必须一个包含另一个
  4. 注释： <!-- -->
  5. 单双引号均表示字符串,在多个字符串包含时,需要单双引号交替使用
  6. 大小写不严格区分,但需要统一是大&小写,不可大小写混用
