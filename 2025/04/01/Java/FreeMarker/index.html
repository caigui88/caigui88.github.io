3、默认值

FreeMarker 对变量的空值校验是很严格的，如果模板中某个对象为空，FreeMarker 将会报错而导致模板生成中断。

为了防止这个问题，建议给可能为空的参数都设置默认值。使用 表达式!默认值 的语法，示例代码如下：dWCTqTa/WiFmXHl4+q5sKVXVlhR8/dHGRVI7G0kL0bU=

上述代码中，如果 user 对象为空，则会输出 “用户为空” 字符串。
